# Go-Learning
## My Golang learning path - Personal Notes


### What is Go ?

1. Open source - developed by Google.
2. Developed in 2007.
3. Focused on: simplicity, clarity, scalability.
4. High performance.
5. Batteries included.
6. Static typed.

# Start a new Project
go mod init projectname

# Run your program
go run main.go

# Build executable
go build

# Add dependencies automatically
go mod tidy

# Download dependencies
go get github.com/someuser/somepackage

# List all dependencies
go list -m all
